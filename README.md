    <article class="markdown-body entry-content container-lg" itemprop="text"><p>README</p>
<p>NORMA-The network makeup artist: a web tool for network annotation visualization</p>
<p>What is this repository for?</p>
<p>NORMA is a web tool for interactive network annotation visualization and topological analysis, able to handle multiple networks and annotations simultaneously. Precalculated annotations (e.g. Gene Ontology/Pathway enrichment or clustering results) can be uploaded and visualized in a network either as colored pie-chart nodes or as color-filled convex hulls in a Venn-diagram-like style. In the case where no annotation exists, algorithms for automated community detection are offered. Users can adjust the network views using standard layout algorithms or allow NORMA to slightly modify them for visually better group separation. Once a network view is set, users can interactively select and highlight any group of interest in order to generate publication-ready figures. Briefly, with NORMA, users can encode three types of information simultaneously. These are: i) the network, ii) the communities or annotations and iii) node categories or expression values. Finally, NORMA offers basic topological analysis and direct topological comparison across any of the selected networks.</p>
<p>How do I get set it up?</p>
<p>Install R,
Install Rstudio,
Install libraries in the global.R file,
Open ui.R or server.R and Run project,</p>
<p>Publication</p>
<p>Has been submitted for publication</p>
<p>How does it work?</p>
<p>Please visit <a href="http://bib.fleming.gr:3838/NORMA/" rel="nofollow">http://bib.fleming.gr:3838/NORMA/</a> to run it online or download the code and run it locally using Rstudio.</p>
<p>What is the input file format?</p>
<p>Examples and instructions regarding the input files can be found in NORMA's Help/About pages (Tab Input File)</p>
<p>Are there any datasets in order to try NORMA out?</p>
<p>You can download various examples from NORMA's Help/About pages (Tab Examples)</p>
